Description

The basic_string_view class better matches what the standards committee is doing and has WAY more constexpr support. Code that uses boost::string_ref should continue to work. I'll be watching the testbots for results, but I believe that the only part of boost that was using basic_string_ref was Boost.Log, and that was switched over to basic_string_view in December.
